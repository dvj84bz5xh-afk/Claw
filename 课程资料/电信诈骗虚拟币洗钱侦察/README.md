## 📝 使用建议

### 教学前准备
1. 熟悉《反诈实战手册》内容，准备话术破解案例
2. 测试区块链浏览器访问（建议使用OKLink）
3. 准备警情模拟场景和角色扮演材料
4. 打印《快速参考指南》供学员课后参考
5. 准备实训室计算机环境（手机取证软件）

### 教学中建议
1. **第1课时**：重点进行警情模拟，让学员掌握接警要领
2. **第2课时**：通过图示讲解区块链原理，降低理解难度
3. **第3课时**：边演示边让学员操作，确保每人都能独立查询
4. **第4课时**：
   - 双轨追踪部分使用真实案例（脱敏）演示
   - 综合演练分组进行，每组4-5人
   - 要求各组绘制双轨追踪图并汇报

### 教学后跟进
1. 收集学员反馈，重点了解双轨追踪方法的掌握情况
2. 建立学员交流群，分享实战经验和案件线索
3. 定期更新《快速参考指南》中的交易所联系方式
4. 关注虚拟币犯罪新趋势，及时补充案例
