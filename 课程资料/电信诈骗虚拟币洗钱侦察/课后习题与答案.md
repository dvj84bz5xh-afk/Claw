## 参考答案

### 第一课时答案

**一、填空题**
1. 刷单返利
2. 去中心化、跨境便捷
3. 虚假网络投资理财
4. 跨链桥、稳定币USDT
5. 匿名性

**二、选择题**
1. C  2. B  3. C  4. C  5. C

**三、简答题**

1. **传统洗钱与虚拟币洗钱的区别**：
   - 传统洗钱：依赖银行体系，有监管中介，跨境转账慢（数天），交易可逆，身份可追踪
   - 虚拟币洗钱：去中心化，无监管中介，跨境秒级转账，交易不可逆，匿名性强
   - 虚拟币洗钱更难追踪，但链上记录永久保存，可通过技术手段分析

2. **三种以上电信网络诈骗作案手法**：
   - 刷单返利：以"点赞赚佣金"为诱饵，先让受害人垫付小额资金，后要求大额投入
   - 杀猪盘：通过婚恋交友建立感情，后诱导投资虚假平台
   - 冒充公检法：假冒警察/检察官，称受害人涉嫌犯罪，要求转账到"安全账户"
   - 虚假投资理财：推荐"内部渠道"高收益投资，初期返利后卷款跑路

### 第二课时答案

**一、填空题**
1. 时间戳、共识机制
2. PoW（工作量证明）、PoS（权益证明）
3. 热、冷
4. 波场（TRON）、1
5. 哈希

**二、选择题**
1. B  2. B  3. C  4. B  5. B

**三、简答题**

1. **公钥和私钥的关系**：
   - 私钥通过椭圆曲线算法生成公钥，公钥通过哈希算法生成地址
   - 私钥可以推导出公钥和地址，但反向不可行
   - 私钥用于签名交易，证明资金所有权；公钥/地址用于接收资金
   - 私钥必须保密，因为掌握私钥就等于掌握对应地址的所有资金

2. **公开透明但匿名的解释**：
   - 公开透明：所有交易记录在区块链上公开可查，任何人都可以查看任意地址的余额和交易历史
   - 匿名：地址本身不包含身份信息，不直接关联真实身份
   - 伪匿名性：虽然地址匿名，但通过交易所KYC、交易行为分析等手段可以关联到真实身份

### 第三课时答案

**一、填空题**
1. 获取涉案交易哈希(TxHash)、追溯资金来源和去向
2. Etherscan、TronScan
3. 零知识、资金池
4. 同一主体
5. UTC（协调世界时）、+8

**二、选择题**
1. B  2. C  3. B  4. B  5. C

**三、简答题**

1. **链上资金追踪六步法**：
   - 第1步：获取涉案交易哈希(TxHash)
   - 第2步：在区块链浏览器查询交易详情
   - 第3步：分析输入输出地址
   - 第4步：标记地址标签（交易所、混币器等）
   - 第5步：绘制资金流向图
   - 第6步：追溯资金来源和去向

2. **识别混币器使用的方法**：
   - 查看交易是否转入已知的混币器合约地址
   - 分析交易特征：资金突然"消失"或"出现"而无明显来源/去向
   - 使用地址标签库识别混币器相关地址
   - 查看金额是否为混币器的固定面额（如Tornado Cash的0.1/1/10/100 ETH）
   - 分析时间模式：短时间内大量等额存取

### 第四课时答案

**一、填空题**
1. 交易哈希、查询时间
2. 双边司法协助条约、金融情报交换
3. 银行流水分析、情报线索
4. 完整、不可篡改
5. 节点类型

**二、选择题**
1. C  2. C  3. C  4. B  5. D

**三、案例分析题**

1. **侦查方案**：
   - **下一步侦查方向**：
     ① 追踪地址A和地址B的后续资金流向
     ② 分析地址交易模式，识别是否为组织化作案
     ③ 查找地址与交易所的关联，获取KYC信息
   - **需要调取的证据**：
     ① 受害人银行转账记录
     ② 涉案银行账户流水
     ③ 区块链交易记录（通过浏览器查询并截图固定）
     ④ 与案件相关的聊天记录、APP截图

2. **调取交易所数据的流程**：
   - **准备法律文书**：
     ① 立案决定书
     ② 调取证据通知书
     ③ 警官证复印件
     ④ 中英文对照的协查函（说明案件情况、调取需求、法律依据）
   - **调取流程**：
     ① 通过官方渠道联系交易所合规部门
     ② 提交法律文书和协查函
     ③ 等待交易所回复（通常1-4周）
     ④ 收到数据后进行分析和证据固定

3. **混币器案件的进一步侦查**：
   - **时间戳分析**：分析资金存入和取出的时间关联
   - **金额匹配**：寻找相同或相近金额的存入/取出交易对
   - **交易所突破**：追踪混币器前资金来源和混币后资金去向，看是否涉及交易所，通过KYC信息关联
   - **行为分析**：分析存取款前后的交易行为，寻找关联线索
   - **前端信息**：如有条件，调取嫌疑人使用混币器时的IP地址、设备信息等
   - **协查通报**：将涉案地址通报其他公安机关，看是否有关联案件
