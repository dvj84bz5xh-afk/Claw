基于 Claude Code 泄露源码，创建了全新的 Agent Core 模块：

```
agent-core/
├── __init__.py              # 模块入口
├── tool_registry.py         # 工具注册表（动态发现、权限控制）
├── session_manager.py       # 会话管理（持久化、历史追踪）
├── permission_context.py    # 权限上下文（三级权限）
├── command_router.py        # 命令路由（/command 风格）
├── tools_snapshot.json      # 工具清单文件
└── example_usage.py         # 使用示例
```

#### 核心改进

1. **工具注册表模式**
   - 工具清单与代码分离（JSON 配置）
   - 动态工具注册
   - 基于 token 的智能路由
   - 13个内置工具：read_file, write_to_file, execute_command, bash 等

2. **三级权限系统**
   - read-only: 只读操作
   - workspace-write: 工作区写入
   - danger-full-access: 完全访问
   - 工具级别的权限控制
   - 支持白名单/黑名单模式

3. **会话持久化**
   - 保存对话历史
   - 记录工具使用情况
   - 流式事件追踪
   - 支持会话恢复 (`--resume latest`)
   - 存储位置: `~/.claw/sessions/`

4. **命令路由器**
   - 支持 /command 风格命令
   - 内置命令: /help, /status, /cost, /doctor, /history
   - 命令别名支持
   - 动态命令注册
