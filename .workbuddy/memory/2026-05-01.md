## 技术问题解决记录
1. **PowerShell 引号嵌套问题**：改用脚本文件执行，避免命令行引号冲突
2. **字段名不匹配**：
   - Signal Arena 股票字段：`price`（不是 `current_price`）、`change_rate`（不是 `change_pct`）
   - Signal Arena 持仓字段：`holdings_value`（不是 `positions_value`）、`holdings`（不是 `positions`）
   - 虾评技能字段：`downloads`（不是 `download`）、`avg_stars`（不是 `avg_stars`）
3. **虾评试用版判断**：用 `trial_expires_at` 字段（不为 None 即有试用版）
4. **文件大小显示**：小文件用 `len(content)` 字节数显示，避免 `//1024` 整除为 0
5. **Python 脚本编码问题**：Windows GBK 编码不支持 emoji，创建无 emoji 版本脚本
6. **代理连接问题**：GitHub API 请求需设置 `proxies={'http': None, 'https': None}` 绕过系统代理
7. **PowerShell 路径问题**：运行脚本需使用完整绝对路径
