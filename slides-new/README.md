# 上海小学英语单元整体教学课件系统

## 📋 项目概述

这是一个基于 Node.js + PptxGenJS 的自动化 PowerPoint 课件生成系统，专门为**上海小学英语四年级下册 Module 3 Unit 3 "Days of the week"**单元设计。课件体现了**素养导向**的教学理念，融合了**数字化教学转型**的创新实践。

## 🎯 核心特点

### 1. 教学设计特点
- **素养导向**: 围绕语言能力、文化意识、思维品质、学习能力四大核心素养
- **单元整体**: 四课时完整教学设计，从情境导入到成果展示形成闭环
- **五育融合**: 德智体美劳全面融入英语教学
- **技术赋能**: 整合"三个助手"数字化教学平台

### 2. 技术实现特点
- **模块化设计**: 每页幻灯片独立 JavaScript 文件，易于维护和扩展
- **主题系统**: 统一配色方案，确保视觉一致性
- **自动化生成**: 一键生成专业级 PowerPoint 课件
- **可定制化**: 所有内容参数化，便于修改和适配

## 🚀 快速开始

### 环境要求
- Node.js 14+ (推荐 v18+)
- npm 或 yarn 包管理器

### 安装依赖
```bash
cd slides-new
npm init -y
npm install pptxgenjs
```

### 生成课件
```bash
# 使用优化版编译脚本
node compile-fixed.js
```

### 查看结果
生成的PPT文件位于 `output/上海小学英语单元整体教学课件.pptx`，打开即可查看12页完整课件。

## 🎨 设计系统

### 配色方案
- **主色 (primary)**: `#2ec4b6` - 青色，用于标题和重要元素
- **辅色 (secondary)**: `#ff9f1c` - 橙色，用于强调内容
- **重点色 (accent)**: `#e71d36` - 红色，用于重要提示
- **浅色 (light)**: `#ffbf69` - 浅橙色，用于辅助元素
- **背景色 (bg)**: `#f2f2f2` - 浅灰色，用于卡片背景

### 字体规范
- **标题**: 微软雅黑，32-40pt，加粗
- **正文**: 微软雅黑/Arial，10-14pt
- **页脚**: 10pt，灰色

## 🔧 自定义修改

### 修改内容
1. **文字内容**: 直接在对应的 `slide-XX.js` 文件中修改文本
2. **布局调整**: 修改 `x, y, w, h` 参数调整位置和大小
3. **颜色调整**: 在 `compile.js` 中修改 `theme` 对象
4. **新增页面**: 创建新的 `slide-13.js` 并在编译脚本中添加加载

### 修改示例
```javascript
// 修改 slide-01.js 中的标题
const slideConfig = {
  title: '你的新标题',
  subtitle: '你的新副标题',
  // ...其他配置
};

// 修改配色主题（在 compile.js 中）
const theme = {
  primary: "你的主色",      // 例如 "4A90E2"
  secondary: "你的辅色",    // 例如 "50E3C2"
  // ...其他颜色
};
```

## 📚 教学理念说明

### 1. 素养导向
基于《义务教育英语课程标准（2025修订版）》，强调：
- **语言能力**: 听说读写综合能力发展
- **文化意识**: 跨文化理解和国际视野
- **思维品质**: 批判性思维和创新能力
- **学习能力**: 自主学习和合作学习能力

### 2. 三个助手平台
上海教育数字化转型的核心：
- **教学助手**: AI备课、AR/VR情境创设、个性化学习路径
- **评价检测助手**: 过程性数据采集、AI语音测评、成长档案
- **作业助手**: 智能作业批改、分层作业推送、家校协同

### 3. 五育融合
将德育、智育、体育、美育、劳育融入英语教学：
- **德育**: 榜样教育、价值观培养
- **智育**: 知识学习、思维训练
- **体育**: 健康意识、运动习惯
- **美育**: 审美情趣、艺术表达
- **劳育**: 实践能力、劳动意识

## 🛠 技术实现细节

### PptxGenJS API 使用
```javascript
// 添加形状
slide.addShape(pres.SHAPE_TYPE.ROUNDED_RECTANGLE, {
  x: 0.5, y: 1.0, w: 4.0, h: 2.0,
  fill: { color: theme.primary },
  line: { color: "FFFFFF", width: 2 }
});

// 添加文本
slide.addText("标题", {
  x: 0.5, y: 1.0, w: 4.0, h: 0.5,
  fontSize: 24, bold: true,
  color: theme.primary,
  align: 'center'
});
```

### 模块化架构
每个幻灯片模块都遵循相同结构：
```javascript
function createSlide(pres, theme) {
  const slide = pres.addSlide();
  // ... 幻灯片内容
  return slide;
}

module.exports = { createSlide };
```
